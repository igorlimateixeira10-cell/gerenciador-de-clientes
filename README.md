# Gerenciador de Clientes

Projeto desenvolvido como parte do curso da EBAC, utilizando HTML5, CSS3 (com design futurista e Glassmorphism) e JavaScript (Fetch API para consumo do CrudCrud).

## Como rodar o projeto
1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Certifique-se de estar conectado à internet para consumir os dados da API CrudCrud.

## Tecnologias
- HTML5, CSS3, JavaScript.
